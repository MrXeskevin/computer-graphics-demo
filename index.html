HTML5 Canvas: Intro Computer Graphics Concepts

Primitives → Composition → Transformations (translate/rotate/scale) → Animation (requestAnimationFrame)

Hover over the canvas to “highlight” an object (scales slightly) to reinforce transforms.
House: primitives + composition
Car: translation + wheel rotation
Person: scaling + slight translation (“breathing”)

What you’re seeing

1) Geometric primitives

  • Rectangles (house body, doors, car body, windows, torso)
  • Triangles (roof)
  • Circles (wheels, head)
  • Lines (arms/legs)

2) Object composition

Each object (house/car/person) is built from smaller shapes in its own drawHouse, drawCar, and drawPerson function.

3) Transformations

  • Translation: the car moves across the canvas using ctx.translate().
  • Rotation: wheels rotate using ctx.rotate().
  • Scaling: the person scales subtly with ctx.scale().

4) Animation loop

A standard requestAnimationFrame() loop updates positions/angles then redraws the whole scene.

Runs locally: open index.html. No build step.
